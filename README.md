# Trade Dashboard Components

This repository contains reusable React components for trade education and visualization. The `MESChartExplainerVideo` component demonstrates breakout, rejection, and reclaim scenarios on a miniature candlestick chart with animated playback controls.

## Available Components

- `MESChartExplainerVideo` – interactive explainer highlighting different MES trading scenarios. The component is located at [`src/components/MESChartExplainerVideo.tsx`](src/components/MESChartExplainerVideo.tsx).

## Development

Install dependencies and run the TypeScript type check:

```bash
npm install
npm run lint
```

The project uses a path alias (`@/`) that maps to the `src/` directory, configured in `tsconfig.json`.
