# Trade Dashboard

A lightweight, static preview of a trading dashboard that you can open locally or host with GitHub Pages.

## View the dashboard

To preview the dashboard UI, open the HTML file directly in your browser:

- [View the dashboard](docs/index.html)

If you are using GitHub Pages, you can publish the `docs/` folder to expose the same view publicly.

## Customising the data

The sample data in `docs/index.html` is intentionally static so you can replace it with your own portfolio metrics, chart series, and recent trade history. Search for the `equityChart` dataset and the `<tbody>` rows to plug in your live feed.
